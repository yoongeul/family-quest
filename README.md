# 우리집 오늘의 미션 — 1단계 MVP

기능
- 다연/유나 전환
- 오늘 일정 체크
- 월~금 주간 보기
- 엄마가 반복 일정 추가/삭제
- '용돈 조건' 일정 표시
- 브라우저 로컬 저장
- JSON 백업/복원
- 홈 화면 추가 가능한 PWA 구조

## 실행
가장 간단한 방법:
1. 이 폴더에서 간단한 웹 서버를 실행합니다.
2. 브라우저에서 접속합니다.

예:
python3 -m http.server 8000

그다음 http://localhost:8000 에 접속합니다.

주의
현재 버전은 기기별 로컬 저장이라 아이폰과 갤럭시 사이에 자동 동기화되지 않습니다.
가족 여러 기기에서 함께 쓰려면 다음 단계에서 Firebase 또는 Supabase 로그인을 연결해야 합니다.
